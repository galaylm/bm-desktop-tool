# Changelog

## 1.7.3 - 2026-06-22

- 修复扩展管理里删除 OKX / 已导入扩展后，实例重启又自动恢复的问题：删除扩展时会同步解绑各实例 `--load-extension` 启动参数。
- 清理实例配置和浏览器 Profile 内残留的扩展数据，包括 Preferences 里的扩展 settings 与 pinned_extensions，避免旧扩展继续被 Chromium 加载。
- 如果导入扩展目录已不再被任何实例引用，会自动清理对应的 `extensions/imported` 目录，避免旧扩展目录残留造成误判。

## 1.7.2 - 2026-06-17

- 修复部分机器在启动实例后偶发整体闪退的问题：移除实例级扩展弹窗尺寸巡检器，避免与 Chromium 扩展弹窗/DevTools 窗口生命周期竞争导致进程异常退出。
- 修复窗口同步顶部悬浮条在失焦/收起时频繁闪烁、跳动的问题：紧凑面板不再在每次状态切换时重复 `WindowShow()` / `WindowUnminimise()`，只在首次启动时执行一次窗口唤起。
- 修复窗口同步紧凑悬浮条底部偶发黑边的问题：移除内容尺寸外额外的透明宿主留边，避免无边框窗口底部露出宿主背景。
- 优化窗口同步悬浮条的失焦视觉状态：未激活时去掉额外阴影与模糊，降低透明 WebView2 在失焦态下的抖动感。
- 保留窗口同步独立面板的明确关闭入口，同时继续限制 sync-panel 启动初期的误关闭路径，避免被误判为“主程序闪退”。
- 补强窗口同步独立面板的 hover 稳定性与交互切换体验，使顶部内容型控制条更接近固定悬浮工具条效果。

## 1.5.6 - 2026-06-08

- 修复网站连接扩展时「登录请求」类弹窗被 Chromium 放大成浏览器窗口大小的问题：现在会按扩展弹窗尺寸收敛到约 390x620。
- 补强通用连接/登录请求标题识别，避免只识别具体钱包品牌导致非钱包品牌或中文请求弹窗漏处理。

## 1.5.5 - 2026-06-08

- 为窗口同步独立面板补回内置关闭按钮，适配无边框窗口没有系统标题栏按钮的问题。

## 1.5.4 - 2026-06-08

- 调整窗口同步独立面板为深色不透明无边框宿主，避开透明 WebView2 面板在部分环境下闪退/闪烁关闭。
- 增加窗口同步面板启动与重复实例退出生命周期日志，便于定位真实退出路径。

## 1.5.3 - 2026-06-08

- 修复窗口同步器独立面板启动后几秒自动正常退出的问题，阻止 sync-panel 未授权 close 路径。
- 调整 sync-panel 生命周期：面板进程退出不再影响主程序托盘，前端移除会直接触发 Quit 的关闭按钮。

## 1.5.2 - 2026-06-07

- 修复 Windows 托盘宿主窗样式：将 systray 隐藏宿主窗改为 ToolWindow + NoActivate + Popup，彻底避免 `GDI+ Window` 出现在任务栏预览、Alt-Tab 和悬浮缩略图中。
- 本地接管 `github.com/energye/systray` Windows 实现，构建时固定使用项目内修正版，避免后续依赖回退导致幽灵窗口问题复发。

## 1.5.1 - 2026-06-06

- 修复部分实例启动时误报“浏览器进程在完成就绪前退出”的问题：启动前会先检查同一 user-data-dir 是否仍被现有浏览器进程占用。
- 新增 Chromium Singleton 残留锁自愈：无活进程占用时会自动清理 SingletonLock / SingletonCookie / SingletonSocket，避免旧锁阻止实例再次启动。
- 占用冲突时直接给出更明确的用户目录占用提示，减少被误判为内核损坏、参数错误或安全软件拦截。

## 1.5.0 - 2026-06-06

- 修复扩展图标点击后弹窗不显示的问题：补上通用扩展窗口恢复逻辑，设置页与非钱包类扩展弹窗也能正常回到屏幕内显示。
- 避免误判主浏览器窗口为扩展弹窗：补强 Chromium / Chrome / Google Chrome 标题保护，防止主标签页被错误移出屏幕。
- 清理测试环境中误带入的日本代理节点，代理池默认仅保留内置直连与本地代理。

## 1.1.0 - 2026-03-19

- 完善 Linux 支持：补齐 Linux 环境下的开发、打包、安装、启动与运行链路，并持续修复安装版启动与退出稳定性问题。
- 新增 SOCKS 代理测试支持：SOCKS 代理能力已进入测试阶段，后续会继续验证稳定性与兼容性。
- 实验性支持接口触发浏览器：支持通过接口启动浏览器实例，便于后续接入自动化流程。
